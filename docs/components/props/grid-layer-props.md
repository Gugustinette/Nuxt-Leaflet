| opacity     | Opacity of the tiles                                                                                                                                                                                                             | Number                                                        | -        | 1.0       |

| maxZoom     | The maximum zoom level up to which this layer will be displayed (inclusive)                                                                                                                                                      | Number                                                        | -        | undefined |
